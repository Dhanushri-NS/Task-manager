Task Manager

Tasks | About

This app is a simple and user-friendly task management tool. You can add new tasks and update their status as completed using the "Mark Done" option.

This Task Manager app allows users to:

  • Create and manage daily tasks
  • Mark tasks as completed
  • Track what has been accomplished

Built with Angular as part of a hands-on learning journey.
